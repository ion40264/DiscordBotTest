ほっこり茶屋

ID：972700
【方針】
50位以内を目指しつつ、楽しくプレイ！
ガチ勢も、マイペース勢もどちらも楽しめるよう作戦を立てて戦います。

【同盟戦について】
  • チーム制を採用→ 早期アタック型・中期アタック型・最終日特化型の3チームに分かれ、 各スタイルに合わせた作戦を行います。
    → 作戦は同盟主が指示するので、難しく考えなくても大丈夫です！
  • リアル優先OK→ 各日のポイント消費の目安はありますが、忙しい時は無理せず調整可能です！
【募集要項】
・コミュニケーションが取れる方（戦力を教えて頂き作戦を立てるため）
・傭兵歓迎！ 雰囲気を見てから決めたい方もOK！
・申請時のお願い
　→ プロフィールに火力を記入の上、申請をお願いします！
（例：対闇 lv250 1p / lv300 4p）
【一言】
週末にしっかりプレイしたい方や、深夜に活発に動きたい方にもおすすめです！
楽しくワイワイしつつ、しっかり戦う！ 和やかな雰囲気の中で、一緒に楽しく戦いませんか？
